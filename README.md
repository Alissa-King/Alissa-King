### My Projects
Here are the projects I've worked on, showcasing my skills in data analysis, machine learning, and application development:

1. **[BEACON](https://github.com/Alissa-King/BEACON)**  
   An explainable ML decision-support system for predicting financial distress in U.S. housing and human-service nonprofits. Uses a calibrated XGBoost model with SHAP explainability and a governance response matrix (BEAM) to produce actionable distress scores. DBA dissertation artifact.  
   **Technologies Used:** Python, XGBoost, SHAP, Scikit-learn, Pandas

2. **[Algo Backtester](https://github.com/Alissa-King/algo-backtester)**  
   A full-stack strategy backtesting platform featuring a custom vectorized engine, walk-forward validation, multi-asset portfolio support, and an interactive Plotly dashboard. Built for rigorous evaluation of quantitative trading strategies.  
   **Technologies Used:** Python, FastAPI, Pandas, Plotly

3. **[Document RAG App](https://github.com/Alissa-King/doc-rag-app)**  
   A retrieval-augmented generation (RAG) app that lets users chat with their own documents. Built with Streamlit as the front end and powered by the Anthropic API for intelligent document Q&A.  
   **Technologies Used:** Python, Streamlit, Anthropic API

4. **[990 Analyzer](https://github.com/Alissa-King/990-Analyzer)**  
   A Flask web application that analyzes the financial health of U.S. nonprofits using IRS Form 990 data from the ProPublica Nonprofit Explorer API. Generates health scores, program spending ratios, operating reserves, revenue trends, and interactive visualizations.  
   **Technologies Used:** Python, Flask, ProPublica API, HTML, CSS

5. **[Model Governance Kit](https://github.com/Alissa-King/model-governance-kit)**  
   A practical toolkit of governance templates and automated checks for predictive and generative models, covering bias audits, performance monitoring, and documentation standards for responsible AI deployment.  
   **Technologies Used:** Python, Markdown

6. **[Economic Indicators Dashboard](https://github.com/Alissa-King/Economic-Indicators-Dashboard)**  
   An interactive dashboard that displays key economic indicators fetched from the Federal Reserve Economic Data (FRED) API. Visualizes GDP, Unemployment Rate, Inflation Rate, and Federal Funds Rate, enabling users to track economic trends over time.  
   **Technologies Used:** Python, Dash, FRED API

7. **[Crypto Screener](https://github.com/Alissa-King/crypto-screener)**  
   A live cryptocurrency research dashboard displaying real-time market data via the CoinGecko API. Features a sortable market table, trending coins, top gainers and losers, and detailed candlestick (OHLC) charts — no paid API key required.  
   **Technologies Used:** Python, Streamlit, CoinGecko API

8. **[Employee Retention Prediction](https://github.com/Alissa-King/Employee-Retention-Prediction)**  
   Uses machine learning techniques to predict employee turnover risk and develop retention strategies. Analyzes the IBM HR Analytics dataset, implementing predictive models to identify attrition factors and provide actionable HR recommendations.  
   **Technologies Used:** Python, Scikit-learn, Pandas

9. **[Finance Dashboard](https://github.com/Alissa-King/Finance-Dashboard)**  
   A Streamlit web application for tracking and managing key financial tasks and deadlines, with a SQLAlchemy-backed database and multi-page navigation for organized financial oversight.  
   **Technologies Used:** Python, Streamlit, SQLAlchemy

10. **[Graduate Program Success Analysis](https://github.com/Alissa-King/Graduate-Program-Success-Analysis)**  
    Analyzes success factors in online graduate education using the Department of Education's College Scorecard dataset. Examines employment rates, earnings, and geographic patterns across over 28,000 graduate programs to provide insights for institutions.  
    **Technologies Used:** Python, Pandas, Matplotlib
